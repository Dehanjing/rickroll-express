## Installation

<h4>
  Setup on termux
</h4>

```bash
$ pkg upgrade && pkg update
$ pkg install git nodejs
$ git clone https://github.com/Dehanjing/rickroll-express rickroll
$ cd rickroll && npm i
$ npm run dev
```
